# PyTEG
**P**ython **T**ext **Ex**traction **G**UI. 

The goal is to build a general purpose web GUI that allows to convert nearly any file type to text. 

Requirements: 
- Containerized docker application
- Web GUI (e.g. created in Django)
- Extracts texts from a broad range of filetypes 
  - Audio (.m4a, .flac, .mp3, .mp4, .wav, .wma, .aac)
  - Images (.png, .jpeg, .gif,...)
  - PDF (.pdf)
  - Spreadsheets (.xlsx, .xls, .ods,...)
  - Documents (.doc, .docx, .odt, ...)
  - ...
